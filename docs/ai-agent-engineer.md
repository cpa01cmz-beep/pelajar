# AI Agent Engineer

## Domain Overview

The AI Agent Engineer specialist focuses on improving the AI agent orchestration system, agent configurations, and workflow efficiency within the repository.

## Responsibilities

- Maintain and improve AI agent configurations
- Optimize agent workflow definitions
- Create and manage agent skills and tools
- Ensure agents work efficiently together
- Monitor agent performance and output quality

## Current State

- Repository uses GitHub Actions with OpenCode for AI agent orchestration
- Multiple specialist agents run in parallel (RnD, Product-Architect, ai-agent-engineer, etc.)
- Workflow: Architect → Specialists → Kritikus → Fixer → PR-Handler
