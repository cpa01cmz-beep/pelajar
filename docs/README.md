# Santri/Pelajar

A fully autonomous system orchestrator that builds and maintains a "Santri/Pelajar" fullstack serverless production-grade application.

## Overview

This repository uses an AI-driven development approach with multiple specialized agents that work together to build and maintain the application. The system is designed to be self-evolving and self-improving.

## Tech Stack

- **Runtime**: Node.js 20
- **Automation**: OpenCode AI agents
- **CI/CD**: GitHub Actions
- **Infrastructure**: Serverless (Cloudflare, Vercel)

## Getting Started

This project is designed to run autonomously via GitHub Actions. To manually trigger a development cycle:

1. Ensure all required secrets are configured in GitHub
2. Navigate to Actions tab
3. Run the "parallel" workflow
